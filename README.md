# Holyrics-for-Linux
Holyrics para distribuciones GNU/Linux, este proyecto NO MODIFICA, NO CAMBIA, NO REESCRIBE NADA, solo empaqueta paquetes para distribuciones GNU/Linux.

Se planea soporte a paquetes:

.deb (Debian + derivadas, Ubuntu + derivadas, Devuan + derivadas)
.rpm (Fedora + derivadas, OpenMandriva, Mageia, ALT Linux, RHEL, Rocky Linux, openSUSE & SUSE)
Pacman (Arch + derivadas) y Chaotic AUR
.xbps (Void Linux)
.apk (Alpine Linux)
.appimage (AppImage)
